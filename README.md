# Twitter Stock Market Analysis

## Project Overview
This project analyzes Twitter stock market data using Data Analytics techniques such as:
- Data Cleaning
- Exploratory Data Analysis
- Statistical Testing
- Visualization
- Dashboard Creation

- ## Dataset

[Download TWTR Dataset](./TWTR.xlsx)

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Dash

## Dataset Columns
- Date
- Open
- High
- Low
- Close
- Adj Close
- Volume

## Features
- Statistical Analysis
- T-Test & Chi-Square Test
- Stock Trend Visualization
- Interactive Dashboard
- Word Cloud Analysis

## Dashboard
Interactive dashboard created using Plotly Dash.


## Conclusion
The project successfully analyzed Twitter stock market trends, volatility, and investor behavior using data analytics techniques.
