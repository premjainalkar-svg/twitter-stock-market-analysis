# Twitter Stock Market Analysis

## Project Overview
This project analyzes Twitter stock market data using Data Analytics techniques such as:
- Data Cleaning
- Exploratory Data Analysis
- Statistical Testing
- Visualization
- Dashboard Creation

- ## Dataset

[Download TWTR Dataset](./TWTR.csv.xlsx)

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Dash

- Stock Trend Visualization
- Interactive Dashboard
- Word Cloud Analysis
